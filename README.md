# Portfolio Évolution

Un portfolio Next.js qui évolue au scroll, montrant différentes étapes de développement web.

## Structure

Le portfolio est divisé en 4 sections qui se révèlent au scroll :

1. **Section 1 - HTML brut** : Affiche le contenu avec les balises HTML visibles
2. **Section 2 - Sans balises** : Contenu sans balises visibles, mais sans CSS ni animations
3. **Section 3 - Avec CSS** : Ajout du style CSS pour un design moderne
4. **Section 4 - Avec animations** : Ajout des animations JavaScript pour une expérience interactive

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```

## Technologies

- Next.js 14
- React 18
- TypeScript
- CSS Modules
